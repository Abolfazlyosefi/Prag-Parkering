
Parkeringssystemet i Prag är en applikation som är utformad för att hantera ett parkeringshus med 100 platser. 
Användare kan parkera fordon (bilar och motorcyklar), hämta dem, flytta dem mellan platser och söka efter fordon med registreringsnummer. 
Systemet hanterar specifika regler för parkering av motorcyklar, vilket tillåter upp till två per plats.

Funktioner: 
Parkera Fordon: Lägg till bilar och motorcyklar på tillgängliga parkeringsplatser.
Hämta Fordon: Ta bort fordon från sina angivna platser.
Flytta Fordon: Ändra platsen för ett parkerat fordon inom garaget.
Sök efter Fordon: Lokalisera fordon med registreringsnummer.
Visa Parkeringsplats: Visa status för en specifik parkeringsplats (upptagen eller tom).
Visa Alla Parkeringsplatser: Se status för alla parkeringsplatser i garaget.
Avsluta: Avsluta applikationen med en bekräftelseprompt.
